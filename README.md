# Muath-TodoLocally
ToDo App store locally for Docker workshop

#### **Open terminal**
1. Clone Repostrie
`git clone https://github.com/GSG-G10/Muath-TodoLocally.git`
2. cd Muath-TodoLocally
3. npm install
4. in terminal `npx json-server --watch 'tmp/data.json' --port 8000`
5. in another termial 
`npm start`
